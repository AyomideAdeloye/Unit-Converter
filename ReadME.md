# Unit Converter

A Python-based tool that performs conversions across multiple measurement types such as length, weight, temperature, and volume.

## Features
- Supports 10+ measurement categories
- Fast and accurate conversions (<1 second per input)
- Eliminates repetitive manual calculations
- Modular design for easy expansion

## Supported Conversions
- Length (meters, kilometers, miles)
- Weight (grams, kilograms, pounds)
- Temperature (Celsius, Fahrenheit)
- Volume (liters, milliliters)

## Technologies Used
- Python

## Purpose
This project was built to automate common unit conversions and improve efficiency compared to manual calculations.

## How to Run
1. Open the project in VS Code or any Python IDE
2. Run the main Python file
3. Input values and select desired conversion

## Example
Input: 10 kilometers → miles  
Output: 6.21 miles
